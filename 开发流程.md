# STM32F103C8T6 PWM 电机转速控制 — 开发流程

## 1. 项目概述

- **MCU**: STM32F103C8T6（Cortex-M3, 64KB Flash, 20KB SRAM, 72MHz）
- **外设**: 直流电机（TB6612/L9110S 驱动）+ OLED 显示 + 按键
- **通信方式**: TIM2 CH3 PWM（1kHz）+ GPIO 方向控制（PA4/PA5）
- **构建工具**: Keil MDK uVision5 / EIDE (VSCode 插件 `cl.eide`)
- **调试器**: ST-Link (SWD)
- **代码仓库**: [GitHub: rjsjing/stm32-pwm-motor](https://github.com/rjsjing/stm32-pwm-motor)

---

## 2. 技术栈

- **MCU**: STM32F103C8T6（Cortex-M3）
- **库**: ST 标准外设库 v3.5.0
- **构建工具**: Keil MDK uVision5 + EIDE (VSCode 插件 `cl.eide`)
- **调试器**: ST-Link (SWD)

---

## 3. 硬件连接

| STM32F103C8T6 | 连接目标 |
|---------------|----------|
| PA2           | 电机 PWM 调速（TIM2_CH3） |
| PA4           | 电机方向 A（正转高） |
| PA5           | 电机方向 B（反转高） |
| PB1           | 按键1（IPU 上拉） |
| PB11          | 按键2（IPU 上拉） |
| PB5           | OLED SCL |
| PB6           | OLED SDA |
| 3.3V          | VCC     |
| GND           | GND     |

## 5. 开发步骤

### 5.1 依赖

- Keil MDK v5 + STM32F10x 设备包
- VSCode + EIDE 插件 (`cl.eide`)
- 编译宏: `USE_STDPERIPH_DRIVER`, `STM32F10X_MD`

### 5.2 编译

```
Keil:  打开 Project.uvprojx → F7 编译
EIDE:  Ctrl+Shift+P → "EIDE: Build"
```

### 5.3 烧录

```
Keil:  F8 烧录（ST-Link SWD）
CLI:   STM32_Programmer_CLI.exe -c port=SWD -w build/Target\ 1/Project.hex
```

### 5.4 功能验证

- 按按键1 电机速度 +20%（0→20→40→...→100→0）
- OLED 实时显示当前转速
